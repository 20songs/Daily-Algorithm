## Commit Message 규칙

첫 추가: [ADD] 본인 이름 + 문제 이름/번호

`git commit -m "[ADD] 김아무개 전기버스"`


수정: [UPDATE] 본인 이름 + 문제 이름/번호 + 수정 사항

`git commit -m "[UPDATE] 김아무개 전기버스 주석 수정했음"`


삭제: [DELETE] 본인 이름 + 문제 이름/번호

`git commit -m "[DELETE] 김아무개 전기버스"`
